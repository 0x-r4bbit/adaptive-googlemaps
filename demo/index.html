Adaptive googlemaps

Directive attributes

  • sensor="false"
  • size="500x300"
  • zoom="6"
  • center="Slovakia"
  • markers="markers"
  • maptype="roadmap"
  • mapevents="{redirect: true, loadmap: false}"

Demo

mapevents: {{{redirect: true, loadmap: false} | json}}

Note: only the first marker is shown in Google Maps redirect

mapevents: no mapevents

mapevents: {{{redirect: false, loadmap: true} | json}}
